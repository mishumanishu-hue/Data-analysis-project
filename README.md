# Data Analysis Project 📊

A data analysis mini project built using Python.
Created by Maneesha | B.Tech AIML | Bahra University

## About This Project
This project analyzes student marks data and creates
visual charts using Python libraries Pandas and Matplotlib.

## Features
- Reads and organizes data using Pandas
- Calculates average marks automatically
- Finds highest scorer in each subject
- Creates colorful bar charts
- Saves chart as image file

## Technologies Used
- Python 3
- Pandas (data organization)
- Matplotlib (chart drawing)

## What This Project Does
1. Takes student marks data
2. Creates a data table
3. Calculates averages
4. Draws a bar chart
5. Saves chart as marks_chart.png

## How to Run
1. Install Python on your computer
2. Install libraries:
   pip install pandas matplotlib
3. Download data_analysis.py
4. Run the file:
   python data_analysis.py
5. A colorful bar chart will appear!

## Output
- Student marks table printed on screen
- Average marks calculated automatically
- Bar chart showing marks of all students
- Chart saved as marks_chart.png
